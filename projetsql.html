Description du projet

Ce projet full-stack relie une interface web à une base de données via un back-end développé en Python. Il illustre le fonctionnement complet d’une application, depuis la saisie des données par l’utilisateur jusqu’à leur stockage et leur affichage dynamique.

Fonctionnalités principales

  • Formulaires connectés à une base de données
  • Traitement des données côté serveur
  • Communication entre le front-end et le back-end
  • Affichage dynamique des informations stockées

Technologies utilisées

HTML, CSS, JavaScript, Python, SQL

Compétences développées

Ce projet m’a permis d’acquérir une vision globale du développement web, de comprendre les échanges entre le front-end, le back-end et la base de données, et de renforcer mes compétences full-stack.
